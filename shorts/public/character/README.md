# 包くん キャラクター画像の置き場

ここに生成したイラストPNGを置く。置いたら「置いた」と伝えれば、動画・アイコン・アウトロに組み込みます。

## ファイル名（この名前で保存してください）

| ファイル名 | 用途 | 必須? |
|---|---|---|
| `baokun.png` | 基本（笑顔）。これだけでもOK | ★必須 |
| `baokun-excited.png` | わくわく（目キラキラ）。単語が出る瞬間に使用 | 任意 |
| `baokun-point.png` | 指さし（教える）。読み上げ中に使用 | 任意 |
| `baokun-wave.png` | 手を振る。アウトロで使用 | 任意 |

## 推奨スペック
- PNG、**背景は透過**が理想（白背景でも切り抜けます）
- キャラ単体（「デイリー中国語」等の文字・赤丸バッジは入れない）
- 正方形に近い構図・中央寄せ・全身（帽子の上〜手まで収まる）
- 4表情はできれば「同じ絵柄で一貫」させる（character sheetで一括生成が楽）

最低 `baokun.png` 1枚あれば動きます。
